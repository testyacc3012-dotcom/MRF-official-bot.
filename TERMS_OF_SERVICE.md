## What This Bot Does

This bot provides moderation tools for Discord servers, including but not limited to banning, kicking, timing out, and warning members. When a moderation action is taken against a user, the bot will:

- Send that user a direct message stating the action taken and the reason given by the moderator.
- Include a link to an appeal/support server in that direct message.
- Optionally post a record of the action to a designated log channel within the server, if one is configured.
